# Xuan

> Paper and ink, Song inspired.

## Install / Build

- **Use as-is**: Copy `theme.css` into your Obsidian themes folder and enable Xuan under Settings → Appearance.
- **Edit locally**: Clone, run `npm install` and `npm run build`. Edit `styles/*.scss` and rebuild to get an updated `theme.css`.

## Simplicity

- **Blank Space** — Paper as canvas, ink as words
- **Ink Wash** — Between dark and light, balance
- **Cinnabar** — A touch of red, not competing
- **Quiet** — Minimalist, lots of whitespace

## Origin

Born from Song aesthetics, minimalist by nature.
Paper as vessel, ink as narrative.
Not competing, yet commanding attention.

## Fonts

The theme does not hardcode fonts; body and headings follow **Settings → Appearance → Font** in Obsidian. To fix fonts in source, edit `--xuan-font-sans` / `--xuan-font-serif` in `styles/_root.scss` and run `npm run build`.
